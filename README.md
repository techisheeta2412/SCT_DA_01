# 📊 Superstore Sales Data Analysis & Interactive Excel Dashboard

## 📌 Project Overview

This project presents an interactive Sales Dashboard developed using Microsoft Excel to analyze Superstore sales data. The dashboard provides a clear overview of sales performance, profitability, quantity sold, and regional performance.

The project focuses on transforming raw sales data into meaningful business insights using Excel data analysis and visualization techniques.

## 🎯 Project Objective

The objective of this project is to analyze sales data and build an interactive dashboard that helps understand business performance across different categories, regions, and years.

## 📊 Key Performance Indicators

- Total Sales: ₹89,51,931.32
- Total Profit: ₹13,12,442.40
- Total Quantity Sold: 130,064
- Profit Margin: 14.66%

## 📈 Dashboard Features

- Sales Analysis by Category
- Sales Analysis by Region
- Profit Analysis by Category
- Interactive Year Slicer
- Interactive Region Slicer
- Dynamic KPI Cards
- Interactive Pivot Charts

## 🔍 Key Insights

- Technology generated the highest sales among all product categories.
- Technology also contributed the highest overall profit.
- The South region recorded comparatively lower sales.
- The dashboard allows users to dynamically filter sales performance by year and region.
